## Instalación

```markdown
git clone https://github.com/Alcatraz2033/xfce-kali-customization.git
cd xfce-kali-customization
./install.sh
```
## Kitty Commands

| Comando de Teclado             | Acción                                                              |
| ------------------------------ | ------------------------------------------------------------------- |
| `Ctrl` + `Shift` + `Enter`     | Divide la terminal en dos.                                          |
| `Ctrl` + `Shift` + `L`         | Reorganiza la distribución de la terminal.                          |
| `Ctrl` + `Shift` + `F`         | Cambia de posición la terminal en uso.                              |
| `Ctrl` + `Shift` + `W`         | Cierra la terminal actual de trabajo.                               |
| `Ctrl` + `Shift` + `Q`         | Cierra todas las terminales.                                        |
| `Ctrl` + `Shift` + `R`         | Ajusta el ancho y largo de la terminal.                             |
| `Ctrl` + `Shift` + `T`         | Abre una nueva pestaña de la terminal.                              |
| `Ctrl` + `Shift` + `Alt` + `T` | Cambia de nombre la pestaña de la terminal.                         |
| `Ctrl`  + `↑`                  | Cambia la terminal en uso hacia la que se encuentra arriba.         |
| `Ctrl`  + `↓`                  | Cambia la terminal en uso hacia la que se encuentra abajo.          |
| `Ctrl`  + `→`                  | Cambia la terminal en uso hacia la que se encuentra a la derecha.   |
| `Ctrl`  + `←`                  | Cambia la terminal en uso hacia la que se encuentra a la izquierda. |
